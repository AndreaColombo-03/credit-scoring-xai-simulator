Interactive Credit Scoring & XAI Web App | Python, XGBoost, SHAP, Streamlit | [Link GitHub/App]
Sviluppato un simulatore web end-to-end per la valutazione del rischio di credito.

Processato un dataset transazionale per ricavare la variabile target tramite Vintage Analysis e implementato un modello XGBoost ottimizzato con cost-sensitive learning per dataset fortemente sbilanciati (1.6% defaults).

Integrata la libreria SHAP per garantire interpretabilità (Explainable AI) conforme alle normative bancarie, rendendo il modello deterministico e testato via
